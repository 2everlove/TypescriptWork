# TypescriptWork

 Learning Typescript by making a various examples
 
 
## Setting
scoop bucket add extras
scoop install nodejs-lts
scoop install touch
npm install touch-cli -g
npm i -g ts-node
npm i -D typescript ts-node

#### As Project creator
1. `npm init --y`
2. `npm i -D typescript ts-node @types/node`
3. `tsc --init`
- `npm run dev`
- `npm run build`

#### As Project user
1. `npm i`
